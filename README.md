Título: Cadastro de Usuário

Descrição: Aplicação de cadastro de usuários que utiliza Fetch API para preencher automaticamente os campos do endereço utilizando o ViaCEP e que salva os dados do formulário em Local Storage, impedindo que o cliente perca a digitação se fechar e abrir o navegador ou recarregar a página.

Tecnologias utilizadas: HTML5, CSS3, JavaScript (Fetch API, JSON)

Pré-requisitos: Navegador atualizado (projeto foi testado e funciona no Google Chrome Versão 142.0.7444.163 64 bits - Caso utilize este navegador, recomenda-se tal versão ou superior). Necessita-se de acesso à Internet, pois a página utiliza uma API externa (VIACEP).

Instalação:

Passos para clonar e rodar: git clone https://github.com/tayanibritto/cadastroUsuario.git cd cadastroUsuario abrir index.html no navegador
Como usar: Preencher os dados e clicar em "Salvar Informações". 
Estrutura do Projeto: index.html scripts.css styles.css

Observações: Por utilizar uma API externa, a página NÃO funcionará corretamente sem acesso à Internet.
